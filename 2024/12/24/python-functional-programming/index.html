python: 函数式编程

函数式编程是一种编程模式，强调下面几个特点：

  1. 编写无副作用的函数（纯函数）：给定相同的输入总是返回相同的输出，并且不产生任何副作用
  2. 无状态函数：应尽量避免依赖任何外部状态或上下文，而是完全依赖输入参数
  3. 函数作为一等公民，使用高阶函数：接受函数作为参数，或返回函数，比如 map、filter、reduce。
  4. 使用不可变数据结构
  5. 递归：递归与不可变数据结构结合，使得代码更具声明性，便于理解
  6. Monads：使用Monads处理副作用（如异常、状态变化、I/O 等）

虽然python不是为函数式编程设计的，但是python支持很多函数式编程的语法。

  1. 在python语法里，函数是一等公民，支持高阶函数，python可以将函数作为参数传递，或者将函数作为返回值

  2. python支持Lambda。

  3. python支持递归

  4. python支持函数式工具（map、filter、reduce），对可迭代对象进行映射、过滤和聚合操作

  5. python支持列表推导式与生成器表达式

  6. python支持不可变数据结构，比如tuple和frozenset

  7. python支持函数式工具模块 (functools)和itertools 模块

  8. python支持闭包与装饰器，闭包允许函数内部定义函数并捕获外部变量；装饰器可以用于对函数进行功能增强。

  9. python支持Immutable Data（如 dataclasses 与 types.MappingProxyType）
