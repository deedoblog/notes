python: 高阶函数

在python语法里，函数是一等公民，支持高阶函数。高阶函数（Higher-Order Function）是指以下两类函数：

  1. 接受一个或多个函数作为参数的函数。
  2. 将一个函数作为结果返回的函数。

函数对象作为函数参数

我们可以将一个函数作为参数传递给另一个函数。例如，将一个函数应用到列表中的每个元素。这时候，lambda经常被用来创建匿名函数传递给另一函数。

Python内置高阶函数

Python 提供了一些常见的高阶函数，例如 map、filter 和 reduce。

  1. map ：将一个函数应用于可迭代对象的每个元素，返回一个包含结果的新迭代器。
  2. filter ：使用一个函数来过滤可迭代对象中的元素，保留返回值为 True 的元素。
  3. reduce ：使用一个二元函数将可迭代对象缩减为单一值。

组合高阶函数

如果我们按照纯函数设计函数，我们可以组合多个高阶函数来实现更复杂的工作流。比如使用Python内置高阶函数实现数据的过滤和累积

总结

高阶函数有许多优点，是 Python 函数式编程的核心，能够帮助开发者编写优雅、简洁且具有表现力的代码

  1. 代码复用性：将逻辑抽象为可重用的函数。
  2. 代码简洁性：关注“做什么”而不是“怎么做”。
  3. 不变性：鼓励函数式编程的原则，比如不变性和无状态性
